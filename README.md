# Basic To-Do Web App

A beginner-friendly to-do web app made with HTML, CSS, and JavaScript.

## Features

- Add daily tasks
- Show pending tasks
- Show completed tasks separately
- Mark pending tasks as complete
- Move completed tasks back to pending
- Edit task text
- Delete tasks
- Clear pending or completed lists
- Show task added and completed timestamps
- Save tasks in browser local storage
- Responsive layout for mobile and desktop

## Files

- `index.html` - App structure and task sections
- `style.css` - App styling, layout, task cards, and responsive design
- `script.js` - Task logic, local storage, edit/delete/complete behavior

## How to Run

Open `index.html` in your browser.

If you use VS Code, you can also run it with the Live Server extension.
